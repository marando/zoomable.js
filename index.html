zoomable.js examples

zoomable.js is a simple way to zoom images with jQuery.

Basic Example

In the below example the zoomable plugin is applied to all <img> elements on the page. The plugin will use each element's src tag as the full-size image. 

							
								<img src="image.jpg">
							
						
							
								$(document).ready(function() {
									$('img').zoomable();
								});

Custom Example

In this example the zoomable plugin is applied to an image of a specific class, and custom plugin options are passed in the jQuery functin call to change the zoomed appearance. 

							
								<img src="image.jpg" class="custom">
							
						
							
								$(document).ready(function() {
									$('img.custom').zoomable({
										padding: '120px',
										bgcolor: '#000000',
										opacity: 0.5,
										blur:    '7px',
										speed:   800,
										border:  '23px solid #000',
										radius:  '23px',
										shadow:  '0 0 23px #000',
									});
								});

Alternage Full Size Image

You can specify an alternate full size image to load by adding a fullsize attribute on the <img> tag: 

							
								<img src="image.jpg" fullsize="image_full.jpg">
							
						
							
								$(document).ready(function() {
									$('img').zoomable();
								});
