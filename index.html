zoomable.js examples

zoomable.js is a simple way to zoom images with jQuery.

Examples

Basic Example

In the below example the zoomable plugin is applied to all <img> elements on the page. The plugin will use each element's src tag as the full-size image. 

								
									<img src="image.jpg">
								
							
								
									$(document).ready(function() {
										$('img').zoomable();
									});

Custom Example

In this example the zoomable plugin is applied to an image of a specific class, and custom plugin options are passed in the jQuery functin call to change the zoomed appearance. 

								
									<img src="image.jpg" class="custom">
								
							
								
									$(document).ready(function() {
										$('img.custom').zoomable({
											padding: '120px',
											bgcolor: '#000000',
											opacity: 0.5,
											blur:    '7px',
											speed:   800,
											border:  '23px solid #000',
											radius:  '23px',
											shadow:  '0 0 23px #000',
										});
									});

Specify Full Size Image

You can specify an alternate full size image to load by adding a fullsize attribute on the <img> tag: 

								
									<img src="image.jpg" fullsize="image_full.jpg">
								
							
								
									$(document).ready(function() {
										$('img').zoomable();
									});

Documentation

Property Sample Value Description
bgcolor #000 background color of the area to the sides of the image
blur 10 px blur applied to elements behind the full size image
border 1px solid #000 border of the full size image
padding 10 px minimum padding between the image and the browser window
radius 2px border radius of fullsize image
shadow 0 0 14px #000 shadow behind the image border
speed 200 speed of fade transitions
